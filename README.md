#La La Lush Marketplace

La La Lush Marketplace is a customer-to-customer (C2C) e-commerce platform developed as part of a final-year Software Engineering project. Inspired by online marketplaces such as Takealot, the platform allows registered users to create listings, browse products, and engage in online buying and selling within a centralized marketplace.

The system was designed to demonstrate key software engineering concepts, including database management, user authentication, product management, and responsive web development. The application was developed using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap.

#Features
User registration and login
Secure authentication system
Product listing and management
Product search and browsing
Customer-to-customer marketplace functionality
Responsive user interface
MySQL database integration
